Задача

Разработать доступную для 3D-печати конструкцию очков виртуальной и дополненной реальности совместимую с большинством смартфонов.

Цели

  • разработать универсальную конструкцию держателя смартфонов;
  • обеспечить удобство подстройки окуляров с учетом физиологических особенностей пользователей;
  • разработать приложения для просмотра панорам.

Статус

Разработка завершена
